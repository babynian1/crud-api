Installation
- Clone repo
- Buat DB
- create table users  :
  CREATE TABLE USERS (
    id int(100),
    nama varchar(100),
    email varchar(100),
    umur varchar(100),
    primary_key(id)
  )
- set db .env
- go run main.go
